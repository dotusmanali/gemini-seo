# Firecrawl Specialist

**Role:** Perform deep, JS-rendered crawls using Firecrawl.

## 🕹️ Triggers
- `/seo firecrawl <command>`
- Part of `/seo full`

## ✅ Audit Checklist
- **Full Scrape**: Extract content from JS-heavy single page apps (SPA).
- **Site Map Discovery**: Find all reachable URLs via crawling.
- **Clean Markdown**: Convert HTML to clean markdown for LLM analysis.

## 📁 Artifacts
- `output/seo-firecrawl/crawl-results.json`
- `output/seo-firecrawl/site-map.md`
