# Contributing to Gemini SEO

First off, thank you for considering contributing to Gemini SEO! It's people like you who make it such a powerful tool for the community.

## 🚀 How Can I Contribute?

### Reporting Bugs
* Check the [issue tracker](https://github.com/dotusmanali/gemini-seo/issues) to see if the bug has already been reported.
* If not, open a new issue. Include a clear title, a detailed description, steps to reproduce, and any relevant logs or screenshots.

### Suggesting Enhancements
* Open an issue with the "enhancement" label.
* Explain the feature, why it's needed, and how it should work.

### Pull Requests
1. **Fork the repo** and create your branch from `main`.
2. **Install dependencies**: `pip install -r requirements.txt`.
3. **Make your changes**. Ensure the code follows the existing style and is well-documented.
4. **Test your changes**: Run any relevant scripts or specialized audits.
5. **Issue a PR**: Describe your changes clearly and link to any related issues.

## 🛠️ Development Setup
* Python 3.10+
* Node.js 18+
* Gemini CLI (latest)

## 📜 Code of Conduct
Please be respectful and professional in all interactions within this project.

## 💎 Rewards
Contributors who land significant features or bug fixes will be featured in the `CONTRIBUTORS.md` file and recognized within the community.

Happy coding!
