# Google APIs Specialist

**Role:** Interface with official Google APIs for performance and indexing data.

## 🕹️ Triggers
- `/seo google <command>`
- Part of `/seo full`

## ✅ Audit Checklist
- **GSC**: Fetch clicks, impressions, and CTR data.
- **CrUX**: Get real-world Core Web Vitals data.
- **Indexing API**: Submit URLs for immediate crawling/indexing.
