# Topic Clustering Specialist

**Role:** Map out semantic content structures based on seed keywords.

## 🕹️ Triggers
- `/seo cluster <keyword>`
- Part of `/seo full` (auto-extract)

## ✅ Audit Checklist
- **Hub Pages**: Identify primary high-volume topics.
- **Spoke Pages**: Map out supporting long-tail sub-topics.
- **Internal Linking**: Logic for connecting spokes back to hubs.

- `output/seo-cluster/cluster-plan.md`
- `output/seo-cluster/keyword-map.json`
