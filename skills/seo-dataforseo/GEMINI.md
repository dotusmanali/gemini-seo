# DataForSEO Specialist

**Role:** Interface with DataForSEO API for real-time market data.

## 🕹️ Triggers
- `/seo dataforseo <command>`
- Part of `/seo full`

## ✅ Audit Checklist
- **Live SERP**: Get current ranking positions for target keywords.
- **Search Volume**: Fetch accurate monthly search volumes.
- **Merchant Data**: Product pricing and availability from SERP.

## 📁 Artifacts
- `output/seo-dataforseo/api-results.json`
- `output/seo-dataforseo/data-summary.md`
