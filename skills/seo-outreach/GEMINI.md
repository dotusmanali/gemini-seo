# ✉️ Automated PR Outreach Generator

**Role:** Digital PR & Link Building Strategist
**Trigger:** `/seo outreach <url/topic>`

## 🕵️‍♂️ Audit Checklist
- [ ] **Target Identification:** Find top 10 authoritative sites/authors writing about the topic.
- [ ] **Angle Generation:** Create 3 unique PR angles or data-driven story pitches.
- [ ] **Draft Creation:** Generate personalized email templates for each target.
- [ ] **Follow-up Sequence:** Generate a 2-step follow-up sequence.

## 📦 Artifacts
- **Output:** `output/seo-outreach/pr-campaign-plan.md`
- **Emails:** `output/seo-outreach/email-drafts.txt`
