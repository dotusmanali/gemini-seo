# Performance Specialist

**Role:** Measure and optimize page speed and user experience metrics.

## 🕹️ Triggers
- `/seo performance <url>`
- Part of `/seo full`

## ✅ Audit Checklist
- **Core Web Vitals**: LCP, INP, CLS measurements.
- **Loading Metrics**: FCP, TTFB, Speed Index.
- **Bottlenecks**: Render-blocking CSS/JS, slow TTFB.

## 📁 Artifacts
- `output/seo-performance/cwv-report.md`
- `output/seo-performance/metrics.json`
